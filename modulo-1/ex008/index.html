Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>.(não semântica)

Nesta frase temos um termo em destaque usando a tag <strong>.(semântica)

Itálido / Enfâse

Nesta frase temos um termo em itálico usando a tag <i>.(não semântica)

Nesta frase temos um termo em enfâse usando a tag <em>.(não semântica)

Texto Marcado / Efeito marca texto

Podemos também criar um texto marcado com a tag <mark>.(não semântica).

Texto Grande e Pequeno

Estamos criando um texto grande e outro texto pequeno neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído, para indicar que deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido, para dar enfâse e indicar que foi adicionado depois.

Exite também texto subinhado, utilizado a tag <u>.(não semântica)

Para indicar uma potência numérica X20+3.

Texto Subscrito
